# Credit Card Fraud Detection

## Objective
Build a machine learning model to classify credit card transactions as fraudulent or legitimate.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Algorithm
- Logistic Regression

## Steps
1. Load dataset
2. Preprocess data
3. Split training/testing data
4. Train Logistic Regression model
5. Evaluate using Accuracy, Confusion Matrix and Classification Report

## Output
The model predicts whether a transaction is fraudulent or legitimate.
